# 案件管理フォルダ・一発ジャンプツール

## 🚀 解決する課題
* **「ファイル探し」の無駄:** 案件ごとに深い階層にあるフォルダを探す手間をゼロにします。
* **保存ミスの防止:** 間違ったフォルダにファイルを保存してしまうヒューマンエラーを構造的に防ぎます。

## ✨ 主な機能
* **1クリック連動:** Accessの案件画面から、対応するWindowsフォルダを直接開きます。
* **クリップボード自動コピー:** ジャンプと同時に「案件番号」や「顧客名」をコピー。ファイル名変更や検索に即座に利用できます。

## 🛠 使用技術
* Access VBA
* Windows API (Shell)
